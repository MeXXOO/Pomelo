# 柚子
    致力于创建开源免费的IM应用，预计支持Windows macOS Android iOS 等平台
    
# 目录结构

#### Pomelo
- protocol-stack (跨平台底层协议封装和通信功能)
- document （文档相关文件）
- TestCase （测试用例等相关测试代码）
- Windows （windows相关工程文件和代码）
- macOS (mac 相关工程文件和代码)
- iOS （iOS 相关工程文件和代码)
- Android (android 相关工程文件和代码)
- Server （server相关代码)
#### 命名约定
```
    为了便于第三方集成和使用，请以pom 作为前缀命名变量和函数，以及类名
```
